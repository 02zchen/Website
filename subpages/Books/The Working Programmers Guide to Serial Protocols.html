The Working Programmers Guide to Serial Protocols

Summary

This book provides information developers need to understand and implement serial file-transfer capabilities. This book has technical information on protocols such as XProtocol, YProtocol, ZProtocol and Kermit. But that is only part of the book, as the other part provides ideas and insight for developers who need to create their own custom protocols. This book is important as it was written in a time where there was no proper documentation that would be available to the masses, along with providing explanations for multiple facets of the serial protocol system. Also, this book includes many examples to help people setup their own serial protocols, which could be useful in some instances.
