The C Programming Language

Summary

The C Programming Language is written by Brian Kernighan and Dennis Richie (the one who designed C). C is one of the most widely used programming languages in the world. Originally used for the Unix operating system, C was made to be compiled in a straightforward way and map efficiently to machine instructions. The book is full of clear and concise programs that many programmers will encounter in everyday life and provided the basics to help new coders easily learn this new language. This book also contains the first version of the “Hello, world.” code, which is now basically every programmer’s first line of code in any language. The C programming language is the basis of many new and popular languages including Python, which is widely used at DreamPort in its systems. This book is integral to new innovations, which is one of DreamPort’s focuses in the programming language department, because to gain new insight, you must go back to your roots and this book lets one do that quickly and easily.
