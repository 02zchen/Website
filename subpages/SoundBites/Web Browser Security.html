Web Browser Security

The web browser is a very common attack vector for malware and other exploits. There is inherent danger in downloading untrusted code and executing it without any third party verification, as well as in talking over the internet. However, you can minimize this risk by hardening your browser and following best practices. First, you should always use an adblocker, such as uBlock Origin, when browsing the internet. Ads are an attack vector due to how they can come from untrusted sources and be embedded within a trusted page. For additional protection, uMatrix or NoScript, tools that block untrusted JavaScript, can be used, though need to be setup for each website. Next, all communication should be done over HTTPS. Extensions such as HTTPS Everywhere can force all traffic that is known to accept HTTPS to use it, even if the website specifies HTTP. Pay attention to browser warnings regarding insecure websites, and double check that pages where you enter credentials or sensitive data are sent over HTTPS. Finally, modern web browsers have a robust auto-updater, but the user still needs to make sure updates happen. In addition, you should make sure only trusted extensions are installed and that all requested permissions make sense.
