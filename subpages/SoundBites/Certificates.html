Certificates

Certificates and digital signing are used in the computer world to help validate programsyou download have not been tampered with. If the computer says no certificate could be found,chances are that someone has replaced the actual program with malicious software before youeven downloaded it. If this is the case, you should not run the program, as it is probablymalware. Certificates are used in almost all software updates, to help verify the integrity of thefiles. Even operating systems like Windows, macOS, and Linux use certificates to check thelegitimacy of updates.There are two parties here, the user, you, and the software developers, Microsoft. Eachparty has a particular key that only works with each other, which is called “asymmetricalencryption”. You have the decryption key, which you downloaded with the rest of the software,and Microsoft has the encryption key, which they encode their data in to keep it secure. OnceMicrosoft encrypts the data, it is then attached to the original product, which is called a digitalsignature. Once your computer decrypts it, it can see that the message was encrypted by Microsoft, verifying the integrity of the files.
